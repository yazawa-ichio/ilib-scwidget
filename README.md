# ilib-scwidget (WIP)
Unity Tool Scripting Widget Package

Unity用のツール用のウィジェット。  
デバッグメニューやGUILayoutでは実装が難しいツールGUIを作るために作成。  
拡張実装としてデバッグメニューも追加しています。  



とりあえずUMPで確認できるように開発中でも公開してます。
